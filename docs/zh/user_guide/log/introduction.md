# 日志参考

## 使用说明

对日志操作进行了归一化，通过六个新环境变量能对MindIE所有组件进行日志设置，参见[表1](#table1)。

**表 1**  日志设置  <a id="table1"></a>

|环境变量|功能|
|--|--|
|MINDIE_LOG_LEVEL|设置日志级别，具体请参见设置日志级别。|
|MINDIE_LOG_TO_FILE|设置日志是否写入文件，具体请参见设置日志展示方式。|
|MINDIE_LOG_TO_STDOUT|设置日志是否打印，具体请参见设置日志展示方式。|
|MINDIE_LOG_PATH|设置日志落盘路径，具体请参见配置日志落盘路径。|
|MINDIE_LOG_VERBOSE|设置日志是否打印可选信息，具体请参见配置日志内容。|
|MINDIE_LOG_ROTATE|设置日志轮转，具体请参见配置日志轮转。|
